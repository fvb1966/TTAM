# TTAM - Table Tennis Academy Manager

Proyecto TTAM — aplicación de escritorio para la gestión de academias y torneos de tenis de mesa.

Estructura inicial y scripts:

- `npm run dev` — arrancar dev server (Vite)
- `npm run build` — build frontend
- `npm start` — ejecutar electron
- `npm run electron:dev` — desarrollo con electron + vite

Principios: offline-first, SQLite + Prisma, Electron + React + TypeScript.
